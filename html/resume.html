The Groupee App

Conceived, built requirements, and developed a web-based application that uses data received from multiple server-side API requests to allow users to login with their Twitter account (OAuth) and automatically display a map of their current location with prompts to purchase tickets on Ticketmaster.com (Ticketmaster API & Google Maps API) to upcoming concerts and shows of the “verified” people they follow (Twitter API). Next Episode

Lead my team through the SDLC, in an agile/lean fashion and implemented feature and bug fixes using the Git branch workflow and pull requests.
